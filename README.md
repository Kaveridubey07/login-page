# 🔐 Login Page

A modern and responsive Login Page built using HTML and CSS. This project focuses on creating a clean user authentication interface with an attractive design, responsive layout, and smooth user experience.

## 🚀 Live Demo

🔗 https://login-page-njao.onrender.com

---

### 📌 Project Overview

The Login Page project is a simple frontend application designed to provide a professional user login interface. It demonstrates modern UI design principles using pure HTML and CSS without any external frameworks or JavaScript.

This project is ideal for beginners learning web development and frontend design concepts.
